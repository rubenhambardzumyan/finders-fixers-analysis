### Direct Instruction Principles for BrainLift Coach

**Core Learning Principles:**
1. **Worked Examples:** Must precede independent practice
2. **Example Quality:** Design quality determines learning effectiveness
3. **Split-Attention Principle:** Requires integrated information presentation
4. **Element Interactivity:** Determines cognitive load impact

**Three Main Components:**
1. **Program Design:**
   - Content analysis
   - Clear communication
   - Instructional formats
   - Skill sequencing
   - Teaching generalizable strategies (not rote memorization)

2. **Organization of Instruction:**
   - Track-based organization solves 'teach and abandon' problem
   - Students grouped by zone of proximal development
   - Flexible grouping (not rigid tracking)
   - Academic learning time as key success predictor

3. **Student-Teacher Interaction:**
   - Scripts specify exact teacher wording and corrections
   - Continuous assessment and feedback
   - Immediate correction and reinforcement

**Five Juxtaposition Principles:**
- Guide clear communication and concept formation
- Each principle addresses a specific source of potential misunderstanding
- Weaknesses in any element cannot be compensated by strengths in others

### DOK3 Insights - Cognitive Architecture & Learning

- **Capacity:** 4 novel elements maximum

**Scaffolding Progression:**
- Must follow progression from complete examples to independent creation
- Removing scaffolding too early causes reversion to aimless content generation
- Present variation only after 60% accuracy on similar examples
- Different examples from different domains overwhelm working memory before concept mastery

**Element Interactivity:**
- High element interactivity when learning several components simultaneously
- Determines cognitive load impact on learning effectiveness
- Split-attention effects depend on element interactivity level
- High element interactivity materials already saturate working memory

**Interleaving & Variation:**
- **Timing:** Builds problem-type discrimination after 50% accuracy achieved
- **Condition:** Present variation only after 60% accuracy on similar examples
- **Risk:** Too early variation overwhelms working memory

**Dual-Channel Processing:**
- Increases total cognitive capacity
- Temporal contiguity principle for words and pictures
- Integration removes extraneous cognitive load

### Direct Instruction Track Structure Analysis

**Current Curriculum Problem:**
- DOK levels taught as isolated units with no practice between introductions
- Working memory's 4-element capacity + high element interactivity = cognitive overload guaranteed
- 'Teach and abandon' pattern where skills disappear from instruction

**Solution:**
- Need distributed reinforcement across 15+ lessons rather than massed introduction
- Track-based organization maintains continuous reinforcement
- Grouping focuses on specific skill readiness rather than global characteristics

### Split-Attention Effects in Learning

**Core Principle:**
- Effects depend on element interactivity level
- High element interactivity materials already saturate working memory
- Integration removes extraneous cognitive load

**Temporal Contiguity:**
- Words and pictures must be presented together temporally
- Reduces need to hold information in working memory while processing related information
